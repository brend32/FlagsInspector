# FlagsInspector
Позволяет наблюдать изменения Flags и SceneFlags

## Установка
1. Скачать и установить [ModTool](https://github.com/brend32/TinyBunny_ModTool) 
2. Создать папку mod (game/mod/) и закинуть папку flags_inspector
